# Morse Coder

Morse Coder builds web tools for translating, decoding, hearing, and learning Morse code.

![Morse Coder icon](morse-coder-icon.png)

## Official Links

- Website: https://morse-coder.com
- Public feedback: https://github.com/morse-coder-com/morse-coder/issues
- Roadmap: https://github.com/morse-coder-com/morse-coder/blob/main/ROADMAP.md
- Support: support@morse-coder.com

## Product Areas

- Morse translator and decoder
- Text, image, and audio decode workflows
- Morse playback and timing tools
- Learning references and practice sheets
- Multilingual Morse education content

This organization hosts public product documentation and feedback channels for Morse Coder.
